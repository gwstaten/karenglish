<h3>Word Order</h3>
Karen follows the Subject, Verb, Object (SVO) order for words in a sentence similar to english

<h3>Subject & Possessive Pronouns</h3>
If these pronouns are found before a verb, they act as subject pronouns, if they are found before a noun, they act as possessive pronouns

<h3>Example breakdown</h3>
The video used the example:

ယမံၤမ့ၢ်နီၢ်ဖီ meaning "My name is Paw"

Syllable by syllable this breaks down as ယ (my) မံၤ (name) မ့ၢ် (is) နီၢ် (female name marker) ဖီ (flower / paw)

if(document.readyState != 'loading')
  initPronounsQuiz();
else
  document.addEventListener("DOMContentLoaded", initPronounsQuiz);

function initPronounsQuiz() {
  const quizEl = document.getElementById("pronouns-quiz");
  if (!quizEl) return; // not on this page
  window.pronounsQuiz = new MatchingQuiz(
    "pronouns-quiz",
    [
      { left: "Me", right: "ယၤ" },
      { left: "We", right: "ပ" },
      { left: "You (plural) / Y'all", right: "(န)သု" },
      { left: "You (subject)", right: "န" }
    ],
    () =>
      document
        .getElementById("post-quiz")
        ?.classList.remove("display-none")
  );

  window.checkPronounsQuiz = function () {
    const score = pronounsQuiz.check();
    const result = document.getElementById("pronouns-result");

    result.textContent = `You got ${score} / 4 correct.`;
    result.style.color = score === 4 ? "green" : "red";
  };
}

</script>
