# Module 1, Intro to Sgaw Karen

Module 1, Intro to Sgaw Karen is meant to give an overview of the absolute basics of Sgaw Karen to allow you to begin communicating with people in Karen. It will not go over any complicated grammar but just give you the basic building blocks / foundation needed to continue in your language learning journey! Check out [Unit 1, Lesson 1](unit1/1.md) to get started!
